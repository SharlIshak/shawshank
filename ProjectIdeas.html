    • PROJECT IDEAS

    The management software will have many inbuilt features, that can be used, hence it being more efficient for users, as they can find all the tools necessary from the software application, rather than using other external applications. The software will have features such as, messaging system both private and public, where each user will be known by a unique email address upon registration. The messaging system will be robust and reliable with fast delivery, the messaging system will have a feature to both audio and video capabilities to ensure solid communication across the production and on set. Other features include production calendar, timelines and checklists to assist with dates throughout the production. The feature will also notify the user as the deadline approaches closer. Crew management feature will ensure all contacts are in one place, the software will organise the production’s contacts in an easy to use interface, the management system can add, edit and remove details of each member and can also create custom members lists within the management software, keeping crew members info organised. File sharing will also be available, an all-in-one media hub, from documents, like scripts, call sheets, videos, storyboards, etc, the cloud-based system will give control over the library, the administrator can control who can access, modify, view and share files, the cloud storage depends on the plan the owner decides to use, such as the Premium plan which would be $89.99 per month with 100Gb of cloud storage, unlimited projects and being able to invite collaborators who are a part of the production. The system will have a preview feature to playback videos, edit documents and download files to a device from the cloud system, making it an all-in-one software. The design of the software will be modern and simplistic to enable ease of use for all user groups. This software will have a secure login/registration form for new and existing users, the credentials will be stored in a secure encrypted database to prevent unidentified theft. Two-step factor authentication will also be used to ensure that no other entity can easily access a person’s account due to two step verification via email or phone. Other features include, a expenses calculator for the production, it will allow the owner to add expenses during the production of the film, at the end of the media production the expenses will be totalled up with dates and descriptions for each expense, such as expense for renting equipment with relevant figures and dates will allow the user to make decisions and will also assist with the productions budget. The expense calculator will keep track of all expenses to ensure that every dollar is correctly added to the overall expenses for the media product. The expense calculator will only be visible for the creator of a project, unless sharing the expense log which can be exported to excel spreadsheet. Another feature will be to have many personalisation options, such as dark mode and theme settings such as font size, font colour etc. This will give an opportunity for the users to customise their user experience.
